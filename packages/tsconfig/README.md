# @erpgsoft/tsconfig

Coleção de configurações TypeScript compartilhadas.

- `base.json`: configuração padrão para aplicações front/back.
- `react-library.json`: ativa JSX com transform `react-jsx`.
- `node-library.json`: configura emissão para bibliotecas Node.js comuns.

Utilize com `"extends": "@erpgsoft/tsconfig/base.json"` em cada projeto.
